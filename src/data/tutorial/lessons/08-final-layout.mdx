# Final Challenge: Dashboard Layout

Let's put it all together! Build a classic dashboard layout:

- Fixed-width sidebar (200px)
- Flexible content area (1fr)
- Header and footer spanning full width
- Some breathing room (gap)

## Your task

1. Set `grid-template-columns` to `200px 1fr`
2. Make the header span all columns with `grid-column: 1 / -1`
3. Same for the footer

export const markup = `<div class="grid">
  <div class="header">Dashboard</div>
  <div class="sidebar">Menu</div>
  <div class="main">Charts & Data</div>
  <div class="footer">© 2024</div>
</div>`;
