# Named Grid Areas

Instead of line numbers, you can **name areas** of your grid. It's like drawing your layout in CSS:

```css
grid-template-areas:
  "header header"
  "sidebar content"
  "footer footer";
```

Then assign items to areas:

```css
.header { grid-area: header; }
```

## Rules

- Each row is a quoted string
- Names must form rectangles (no L-shapes)
- Use `.` for empty cells
