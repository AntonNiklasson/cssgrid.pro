# The fr Unit

The `fr` unit means **"fraction of available space"**. It's the secret weapon of CSS Grid.

```css
grid-template-columns: 1fr 1fr 1fr;
/*  Equal thirds: each column gets 1 fraction  */

grid-template-columns: 1fr 2fr;
/*  The second column is twice as wide  */
```

Think of it like slicing a pizza: `1fr 2fr 1fr` means "1 slice, 2 slices, 1 slice."

## Why use fr instead of %?

- Automatically accounts for gaps
- More intuitive for proportional layouts
- No math required!
