# Making Items Span Multiple Cells

Items can stretch across multiple columns or rows using the `/` syntax:

```css
grid-column: 1 / 3;   /* from line 1 to line 3 (spans 2 columns) */
grid-column: 1 / -1;  /* from first to last line (full width!) */
```

Or use the `span` keyword:

```css
grid-column: span 2;  /* span 2 columns from current position */
```

## Common patterns

- **Full-width header:** `grid-column: 1 / -1`
- **Span 2 columns:** `grid-column: span 2`
