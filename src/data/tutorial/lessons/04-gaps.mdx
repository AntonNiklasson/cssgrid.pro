# Adding Space with Gaps

The `gap` property adds space **between** grid items â€” not around the edges.

```css
gap: 20px;           /* same gap for rows and columns */
gap: 10px 20px;      /* row-gap  column-gap */
```

## Why gap instead of margins?

- Only appears between items, not on edges
- No margin collapse headaches
- Works perfectly with `fr` units
